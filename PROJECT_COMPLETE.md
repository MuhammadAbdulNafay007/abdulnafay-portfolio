## 🚀 Next Steps (In Order)

### Step 1: Customize Your Information (5 minutes)
```bash
# Edit these files with YOUR information:
1. data/portfolio.ts       # Your name, email, links
2. data/skills.ts          # Your technical skills
3. data/projects.ts        # Your best projects
4. data/experience.ts      # Your work experience
5. data/testimonials.ts    # Testimonials (optional)
```

### Step 2: Add Your Assets (5 minutes)
```bash
1. Add project images to: public/projects/
2. Add resume to:         public/resume.pdf
3. Update image paths in: data/projects.ts
```

### Step 3: Customize Design (2 minutes)
```bash
# Optional: Change colors in app/globals.css
--primary: oklch(0.35 0.15 250);    # Your brand color
--accent: oklch(0.60 0.18 195);     # Your accent color
```

### Step 4: Test Locally (2 minutes)
```bash
pnpm install
pnpm dev
# Visit http://localhost:3000
# Test on mobile, dark mode, all links
```

### Step 5: Deploy to Vercel (2 minutes)
```bash
# Option A: Using Vercel CLI
vercel

# Option B: GitHub + Vercel Dashboard
# Push to GitHub → vercel.com → Connect repo → Deploy
```

**Total Time: ~15-20 minutes**
